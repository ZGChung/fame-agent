## 技术亮点

### TTS 音频生成
支持两种方式：
1. **ElevenLabs API** - 高质量语音（需配置 API Key）
2. **macOS say 命令** - 本地回退方案

```python
# 使用示例
audio_path = await vg._generate_tts("要转语音的文字")
```

### 视频生成命令
```bash
# 从图片生成视频
python3 pipeline.py video generate <content_id>

# 生成带 TTS 音频的视频
python3 pipeline.py video generate-with-tts <content_id>
```

### Ken Burns 特效
- 单图自动应用 zoompan 效果
- 多图支持 fade 转场
