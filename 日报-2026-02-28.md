## 🔧 技术亮点

### 1. TTS 音频生成
- **优先**: ElevenLabs API（高质量）
- **回退**: macOS say 命令（离线可用）
- 自动识别中/英文选择语音

### 2. FFmpeg 视频生成
- 本地处理，无需 API 费用
- 支持多种转场效果（fade/slide/none）
- 竖版 1080x1920 适配社交媒体

### 3. 完整工作流
```
图片 → 视频 → TTS → 音视频合并 → 小红书发布
```
